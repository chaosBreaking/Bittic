安装指南：
1. 找一台电脑，Windows/Linux/Mac 都可以。因为我能跨平台！
2. 安装 nodejs v8.5 以上。到 http://nodejs.org 下载安装最新版即可。
3. 下载本代码库，例如命令行： git clone http://github.com/bittic/ticnode
4. 打开命令行，进入 ticnode 目录： cd ticnode
5. 安装： npm install
6. 运行： node server.js
