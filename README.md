安装：
1. 安装 nodejs v9.11 以上版本。到 http://nodejs.org 下载安装最新版即可。
2. 克隆代码库到本地：
  `git clone https://git.faronear.org/tic/node.server.git`
3. 进入ticnode目录：
  `cd ticnode`
4. 安装依赖： 
	`npm install`
5. 启动运行： 
	`node server.js`
